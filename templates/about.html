XML Scraper is a Flask web application written in Python.
Application renders uploaded XML file and with client parameters renders data in structual way which then can be downloaded as json, excel or csv file.

We recomend poppler for converting pdf into xml. Command: pdftohtml -xml -i file.pdf file.xml.

-xml forms xml page
-i ignores the images
-f xxx from what page you want to start
-l xxx what page you want it to be the last one
