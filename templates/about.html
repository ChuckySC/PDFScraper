XML Scraper is a Flask web application written in Python. Application renders uploaded XML file and with client parameters renders data in structual way which then can be downloaded as json file.

We recomend poppler for converting pdf into xml. Command: pdftohtml -xml -i file.pdf file.xml

Option -xml forms xml page
Option -i ignores the images

Additional options:
-f xxx from what page you want to start
-l xxx what page you want it to be the last one
